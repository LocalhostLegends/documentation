## What is HR Platform?

HR Platform is a comprehensive web portal designed to automate and streamline HR processes for medium and large companies. Our goal is to create a solution that bridges the gap between HR departments and employees, making workforce management efficient and transparent.

## Project Scope

The platform will include:

### Core Modules
- **👥 Employee Management** - Complete employee lifecycle management
- **📝 Recruitment** - Streamlined hiring process
- **📊 Analytics** - Real-time dashboards and reports
- **🔐 Authentication** - Secure role-based access control

### Technical Stack
- **Backend**: NestJS with TypeScript
- **Database**: PostgreSQL with TypeORM
- **Authentication**: JWT + Argon2
- **Documentation**: Docusaurus
- **Deployment**: Docker + GitHub Actions
